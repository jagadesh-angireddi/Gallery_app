# Gallery App

A simple React gallery application that fetches images from the Picsum API and displays them in a clean, responsive grid layout using UseState and UseEffect Hooks.

---

##  Features
- Fetches 12 images per page using Picsum API  
- Pagination support (load next/previous images)  
- Responsive card layout  
- Fast loading using Axios  
- Clean UI with TailwindCSS  

---
## React Hooks
- useState
- useEffect

---

## Installation

Clone the project:

```bash
git clone https://github.com/your-username/gallery-app.git
```

Go to the project folder:

```bash
cd gallery-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
